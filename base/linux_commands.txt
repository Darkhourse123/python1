ls - это показывает все файлы и директории, из той директории в которой вы находитесь
ls - a - это показывает файлы и директории, а также скрытые файлы и директории
ls -l - это показывает более подробную информацию о файлах и директориях
 ls -la- это показывает дополнительную информацию о файлах и директориях ,и о скрытых файлах и директориях тоже
 ls путь до директории - это показывает содержимое директории не заходя туда, через путь.
 ls desktop/pyhtonv1/base - пример

 cd - это переход в домашнюю директорию
 cd ~ - это переход в домашнюю директорию
 cd name - это переходит в директорию name, если такая директория существует в текущей директории
 cd . - это переходитв текущую директорию
 cd .. - это переходит на одну директорию назад
 cd ../.. - это переходит на две директории назад

 mkdir test - создает директорию test в текущей директории
 mkdir путь/test - это создает директорию test по пути
 mkdir test1 test2 test3 - это создаст несколько директории test1 test2 test3 в текущей директории

 touch test.txt - это создание файла test c расширением txt

 rm test - это удаляет файл
 rm -rf test - это удаляет директорию test,пустая или не пустая не имеет значение
rmdir test - это удаляет директорию если она пустая

mv test1.txt test2.txt - это переименовывает файлы и директории
mv test1.txt dirtest - это переместит файл test1.txt в директорию dirtest

history - это показывает все команды, которые вводили в данном терминале 
history 10 - это показывает последние 10 команд в данном терминале.

pwd - это показывает путь до той директории где вы выполнили эту команду

nano test.txt - это откроет файл test.txt в текстовом редакторе nano, если такого файла нет, то создаст его.

cat test.txt - это выведет в терминале содержимое файла test.txt

brew - это системный установщик пакетов (macOS)
apt - это системный установщик пакетов
snap - системный установщик пакетов (в основном для приложений)
sudo - super user do.
